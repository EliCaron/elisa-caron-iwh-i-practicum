# Integrating With HubSpot I: Foundations - Practicum

App Node.js/Express che si collega a HubSpot tramite API per gestire un oggetto
personalizzato ("Pizze"), con proprieta: name, ingredienti, prezzo.

## Setup

1. npm install
2. Crea un file .env (copiando .env.example) e inserisci il tuo token del Private App:
   PRIVATE_APP_ACCESS=il-tuo-token
3. node index.js
4. Apri http://localhost:3000
